## Changes Completed Today

### 1. FX conversion layer

- Added [src/lib/fx.js](/Users/user/Desktop/IT/SimpleSafeBanking/src/lib/fx.js).
- Introduced a shared base-currency model:
  - base currency is fixed to `RON`
  - supported display currencies are `RON`, `EUR`, and `USD`
  - conversion always flows through RON
- Added reusable helpers for:
  - normalizing currency codes
  - default FX rates
  - source-to-target amount conversion
  - display conversion formatting

### 2. Persisted FX settings and lifecycle

- Extended app state in [src/lib/AppContext.jsx](/Users/user/Desktop/IT/SimpleSafeBanking/src/lib/AppContext.jsx) with:

- Added local FX actions:
